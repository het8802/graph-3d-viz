Graph 3D Visualization

A powerful 3D graph visualization library for data structures and algorithms

Interactive Demo

Quick Start

Install the package using npm:

npm install graph-3d-viz

Basic usage example:

import { Graph3D } from 'graph-3d-viz'; const data = { type: 'directed', nodes: [ { id: '1', label: 'Node 1' }, { id: '2', label: 'Node 2' }, ], edges: [ { from: '1', to: '2' }, ], }; const graph = new Graph3D({ container: document.getElementById('graph-container'), data, theme: 'light', nodeSize: 0.3, edgeWidth: 2, }); graph.render();

3D Visualization

Render your graphs in beautiful 3D with smooth animations and interactions.

Multiple Layouts

Support for various graph types including directed, undirected, trees, and DAGs.

Customizable

Easily customize node sizes, colors, labels, and edge styles to match your needs.

Interactive

Rotate, zoom, and pan the graph with intuitive controls. Hover effects for nodes and edges.

Configuration Options

interface GraphConfig { container: HTMLElement; data: GraphData; theme?: 'light' | 'dark'; nodeSize?: number; edgeWidth?: number; onNodeClick?: (node: Node) => void; onEdgeClick?: (edge: Edge) => void; onNodeHover?: (node: Node | null) => void; onEdgeHover?: (edge: Edge | null) => void; }
